Documentation for news API

Articles

This page contains all of the avaliable endpoints on this API

GET /api/topics

Description
- serves an array of all topics
Queries
- []
Example Response
- {"topics":[{"slug":"football","description":"Footie!"}]}

POST /api/topics

Description
- Takes a slug and a description and if topic doesnt exist created a new one and returns it back to you
Queries
- []
Example Response
- {"topic":{"slug":"football","description":"Footie!"}}

GET /api/articles

Description
- serves an array of all articles defaults to sort by date in descending order
Queries
- ["author","topic","sort_by","order"]
Example Response
- {"total_count":13,"articles":[{"title":"Seafood substitutions are increasing","topic":"cooking","author":"weegembump","article_id":1,"created_at":"2018-05-30T15:59:13.341Z","votes":0,"comment_count":6,"article_img_url":"https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"}]}

GET /api/articles/:article_id

Description
- responds with the article when passed the articles id
Queries
- []
Example Response
- {"article":{"title":"Living in the shadow of a great man","topic":"mitch","author":"butter_bridge","body":"I find this existence challenging","created_at":"2018-05-30T15:59:13.341Z","votes":100,"article_img_url":"https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"}}

PATCH /api/articles/:article_id

Description
- responds with the updated article when provided a votes value to increase by and a valid article id
Queries
- []
Example Response
- {"article":{"title":"Living in the shadow of a great man","topic":"mitch","author":"butter_bridge","body":"I find this existence challenging","created_at":"2018-05-30T15:59:13.341Z","votes":101,"article_img_url":"https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"}}

DELETE /api/articles/:article_id

Description
- responds with an no content if correct article id used and deletes relevant comments to the article
Queries
- []
Example Response
- {}

GET /api/articles/:article_id/comments

Description
- responds with all of the comments related to an article id default sorted by created_at ascending. Defaults to a limit of 10 and a default page of 1
Queries
- ["limit","p"]
Example Response
- {"comments":[{"comment_id":5,"body":"I hate streaming noses","votes":0,"author":"icellusedkars","article_id":1,"created_at":"2018-05-30T15:59:13.341Z"}]}

POST /api/articles/:article_id/comments

Description
- responds with the posted comment when provided a username and a comment, and a valid article id.
Queries
- []
Example Response
- {"comment":{"comment_id":5,"body":"I hate streaming noses","votes":0,"author":"icellusedkars","article_id":1,"created_at":"2018-05-30T15:59:13.341Z"}}

GET /api/users

Description
- responds with an array of all of the users
Queries
- []
Example Response
- {"users":[{"username":"butter_bridge","name":"jonny","avatar_url":"https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"}]}

GET /api/users/:username

Description
- responds with a user object
Queries
- []
Example Response
- {"user":{"username":"butter_bridge","name":"jonny","avatar_url":"https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"}}

delete /api/comments/:comment_id

Description
- responds with no content when provided with a correct comment ID to delete
Queries
- []
Example Response
- {}

PACTH /api/comments/:comment_id

Description
- responds with the updated comment when provided with a votes change ({inc_votes:1})
Queries
- []
Example Response
- {"comment":{"comment_id":1,"body":"Oh, I've got compassion running out of my nose, pal! I'm the Sultan of Sentiment!","votes":17,"author":"butter_bridge","article_id":9,"created_at":"2018-05-30T15:59:13.341Z"}}

POST /api/articles

Description
- responds with the new article, with added defaults and a comment coun - requires an author (whose username exists), a title, a body and a topic (that exists as a topic already)
Queries
- []
Example Response
- {"article":{"author":"butter_bridge","title":"butterboy","body":"butter bridge is the most buttery","topic":"mitch","article_img_url":"imgpathforbutterbridge","article_id":14,"votes":0,"comment_count":0,"created_at":"2018-05-30T15:59:13.341Z"}}

GET /api/comments

Description
- responds with an array of all the comments
Queries
- []
Example Response
- {"comments":[{"author":"butter_bridge","body":"butter bridge is the most buttery","article_id":14,"created_at":"2018-05-30T15:59:13.341Z","votes":0,"comment_id":1}]}
